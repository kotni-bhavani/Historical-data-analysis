**Predictive Analytics Using Historical Data**
Overview
This project uses historical sales data to predict future trends using Python, Machine Learning, SQL, and Power BI.
**Objectives**
Analyze historical sales data
Build prediction models
Forecast future sales trends
Create interactive dashboards
Tools Used
Python
Power BI
SQL
Pandas
Scikit-learn
**Features**
Data preprocessing
Sales forecasting
Trend analysis
Dashboard visualization
Predictive modeling
Machine Learning Model
Linear Regression is used to predict future sales based on historical data.
